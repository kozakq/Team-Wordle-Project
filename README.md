# JFX Wordle

## Description
This project is an implentation of the New York Times Wordle game ([link](https://www.nytimes.com/games/wordle/index.html)). Wordle is a strategic game where a random word is chosen and the user must make guesses to what the hidden word is, while getting feedback on each guessed word to make improvements to their guesses and get closer to the goal word. The feedback on each guess includes the validity of the guessed word as well as the correspondence of each letters meaning if a letter was in the right spot, if it is contained inside the goal word but in a different location, or neather. On top of this, the user is able to log in to save and view their statistics on past words. This project is coded in java using the javafx graphics library for the visual aspects of the project.
